1. Open een terminal

2. Clone mijn repository 
```
git clone https://github.com/MariaS20/Blok-Tech 
```

3. Zodra je in de clone van mijn repository zit, installeer npm 
```
npm install
```

4. Start de applicatie
```
node server.js
```
